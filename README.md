# Calendar Project

A modern, responsive calendar application built with **Vue 3**, **TypeScript**, and **Tailwind CSS**. This project demonstrates a full-featured calendar with multiple views and event management capabilities.

## 🚀 Features

- **Multiple Views**: Switch between Day, Week, and Month views seamlessly.
- **Event Management**: Create and view events with start and end times.
- **Dynamic Navigation**: Navigate through months, weeks, and days with ease.
- **Responsive Design**: Styled with Tailwind CSS for a consistent experience across devices.
- **State Management**: Uses Pinia for efficient and type-safe state handling.
- **Time Manipulation**: Powered by Moment.js for accurate date and time calculations.

## 🛠 Tech Stack

- **Framework**: [Vue.js 3](https://v3.vuejs.org/) (Composition API)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [SASS](https://sass-lang.com/)
- **Date & Time**: [Moment.js](https://momentjs.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vue CLI](https://cli.vuejs.org/)

## 📦 Project Setup

This project uses `pnpm` as a package manager.

### Install Dependencies
```bash
pnpm install
```

### Compiles and hot-reloads for development
```bash
pnpm serve
```

### Compiles and minifies for production
```bash
pnpm build
```

### Lints and fixes files
```bash
pnpm lint
```
