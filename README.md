# Calendar Test Project

A Vue.js 3 + TypeScript calendar application.

## Project Setup

```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run serve
```

### Compiles and minifies for production
```bash
npm run build
```

### Lints and fixes files
```bash
npm run lint
```
